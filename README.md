# Uppercase Action
This action converts a text to uppercase

## Inputs
### `text`
**Required** The text to be converted to uppercase.

## Outputs

### `uppercase-text`
The text converted to uppercase.

## Example Usage

```yaml
uses: shimayu22/GitHubActions-Docker@v1.0.0
with:
    text: Hello, World!
```
